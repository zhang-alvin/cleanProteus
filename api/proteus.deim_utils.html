proteus.deim_utils.deim_alg(Uin, m)[source]¶

Basic procedure

  • given $m$, dimension for $F$ reduced basis $mathbf{U}_m$
  • call DEIM algorithm to determine $

ec ho$.

  • build $mathbf{P}$ from $
ho$ as
$$ mathbf{P} = [

ec e_{ ho_1}, ec e_{ ho_2},dots, ec e_{ ho_m}]

$$
  • invert $mathbf{P}^Tmathbf{U}_m$
  • return

ho and $mathbf{P}_F=mathbf{U}_m(mathbf{P}^Tmathbf{U}_m)^{-1}$
